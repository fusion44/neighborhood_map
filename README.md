# Neighborhood Map - A Udacity Nanodegree project
Author: Stefan Stammberger

### How to run
Install nodejs and npm

Run commands:
* Dev Server
    * npm install
    * npm run dev-server
    * Open http://localhost:9000/ in a browser

* Minified Build
    * npm install
    * npm run build
    * open ./dist/index.html in your favorite browser
